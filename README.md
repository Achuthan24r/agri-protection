## Overview
An Arduino-based smart agriculture protection system that detects
animal intrusion, fire hazards, and dry soil conditions.

## Features
- Animal motion detection
- Fire detection alarm
- Automatic irrigation control
- Buzzer alert system
- Expandable to GSM alerts

## Hardware Used
- Arduino UNO
- PIR Sensor
- Flame Sensor
- Soil Moisture Sensor
- Relay Module
- Buzzer

## Working
1. PIR detects animal movement.
2. Flame sensor detects fire.
3. Soil sensor monitors moisture.
4. Relay controls water pump automatically.

## Applications
- Smart farming
- Crop protection
- Automated irrigation
- Rural safety systems

## Future Enhancements
- IoT monitoring
- Mobile app control
- Solar powered system
- Camera-based monitoring
